## Calculo de ROI

### Formula Basica

```
         Beneficios del proyecto - Costos del proyecto
ROI = ────────────────────────────────────────────────── × 100
                   Costos del proyecto
```

### Componentes de Costos

| Categoria | Ejemplos |
|-----------|----------|
| Software | Licencias Power BI, SQL Server |
| Hardware | Servidores, almacenamiento |
| Personal | Consultores, equipo interno |
| Capacitacion | Cursos, certificaciones |
| Mantenimiento | Soporte anual, actualizaciones |

### Componentes de Beneficios

| Categoria | Ejemplos |
|-----------|----------|
| Ahorro tiempo | Automatizacion de reportes |
| Mejor decision | Reduccion errores estrategicos |
| Eficiencia | Optimizacion de procesos |
| Ingresos | Identificacion de oportunidades |
| Reduccion costos | Eliminacion de redundancias |
